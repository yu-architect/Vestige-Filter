# 🛡️ Vestige-Filter: 高速生ソケット同期通信型 ログ収集・階層防御SIEMエージェント

理系ボス（テックリード/CTO）およびインフラ運用者向けの「超高速・堅牢なログアグリゲーター兼セキュリティ防壁」です。
Mac(UNIX)/Windowsのクロスプラットフォームに対応し、生ソケット通信による最下層レイヤーでのログ集約、ハッシュ衝突DoSやSlowloris攻撃への先回り防御、および組織のPlaybook（運用フロー）に直結した自動ルーティング（Ops）を単一のC++バイナリで実現します。

## 🧱 作成コンポーネント・仕様一覧

### 【0番：最若番インフラ（採石場）】
*   **仕様**：Mac/WindowsのOSの壁を突破するクロスプラットフォーム対応の生ソケット（Raw Socket）同期通信。
*   **対ハッカー防御**：接続放置による同期フリーズ（Slowloris攻撃）を防止。`setsockopt`による3秒のタイムアウト（`SO_RCVTIMEO`）およびノンブロッキング通信をソケットレイヤーに組み込み。

### 【1番：ログジェネレーター】
*   **仕様**：実験用の大量ダミーログを限界速度で高速生成。
*   **1-A（確率ツマミ）**：ハッカーの計算（確定性）を2進数の根本から破壊する「サイバー欺瞞（Cyber Deception）」技術。`std::mt19937`疑似乱数を用い、怪しい行動を検知したセッション中のみ確率のツマミを自律変動（0〜100の整数判定）させ、5枚の動的防御壁を構築。

### 【2番：仕分け・集約（本尊・本体）】
*   **仕様**：生ソケットから流れてきたログを最速 O(1) の `std::unordered_map` に吸い込ませ、User ID や IP アドレスをキーにして集約。
*   **2-3（件数カウント）**：同一ログを1つにまとめ、右側に発生件数のカウンター（`int count++`）をドッキング。人間の運用コスト（アラート疲れ）を最大まで削減。
*   **対ハッカー防御（穴①）**：ランダム偽ID爆撃によるメモリ枯渇（DoS）を防ぐため、ハッシュマップの `size()` に上限（容量リミッター）を設計。
*   **対ハッカー防御（穴⑤）**：C++標準ハッシュ関数の癖を突いた衝突攻撃（コリジョンDoS）を無効化するため、起動時に生成したランダムシード（ソルト）を混入するカスタムハッシュ関数を実装。

### 【2-A：時限式防壁（トークン）】
*   **仕様**：正社員・契約社員・ハッカーの権限テーブル管理。一時的な契約社員には有効期限（エポックタイム）を付与し、期限切れの1ナノ秒後に「1枚目の壁（アクセス拒否）」へ即時フォールバック。
*   **対ハッカー防御（穴②）**：パケット内の過去時間偽装（タイムジャック）を完全に無効化。クライアント時間は1ミリも信用せず、100%絶対に「収集サーバー側の内部時計（`std::chrono`）」のローカル時間のみで厳密判定。

### 【3番：自動ルーティング（交通整理）】
判定基準を『点（IP/ID）』『面（地域）』『エラーの深さ（内容）』で厳密に切り分け、人間の組織の運用フロー（Playbook）へ自動振り分けを行います。



| 警戒レベル | 判定基準（点・面・深さ） | システム・組織のOpsアクション |
| :--- | :--- | :--- |
| 🔴 CRITICAL | ・同一の場所（IP/MAC/ID）からの異常な回数<br>・過去のブラックリスト判定に合致<br>・深いパラメータへの攻撃 | ・セキュリティ部門へ最高優先度で即時自動アラート<br>・当該セッションの2-Aトークンを即座に自動無効化（アカウント爆破） |
| 🟢 MEDIUM | ・特定の地域（国やIP帯）からの集中的なエラー | ・サイバー攻撃ではなく「回線障害やバグ」と判定<br>・インフラ（システム）部門のタスク管理へ平和に自動パス |
| 🟡 WARN | ・不特定多数からの「初見のエラー（未知のログ）」<br>・まばらに発生する同一のエラー | ・件数や中身に応じてレベル1〜3へ動的スコアリング<br>・**レベル3（穴③防御）**：WARN全体の総件数がしきい値を超えた、または不穏な文字列を検知した場合、ステルス攻撃（ゼロデイ）とみなし自動で 🔴 CRITICAL へ昇格。 |

## 🛠️ 開発環境 / クイックスタート
*   **言語**：C++17以上
*   **コンパイラ**：Clang (macOS) / MSVC (Windows)
*   **ビルドシステム**：CMake 3.15+

```bash
# リポジトリのクローン
git clone https://github.com
cd Vestige-Filter

# ビルド
mkdir build && cd build
cmake ..
cmake --build .
```
